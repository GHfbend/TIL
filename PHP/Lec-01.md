# PHP 기초문법

### PHP 모드 설정

 1. <? echo("가장 간단한 방법입니다.\n"); ?>
 2. <?php echo("XML을 같이 사용한다면 이렇게\n"); ?>
 3. <script language="php">
      echo("일부 에디터에서는 적용 안됨\n");
    </script>
 4. <% echo("ASP style"); %>
    <%= $variable; # 다음과 같습니다. <% echo .." %>

3번과 4번은 PHP 컴파일시 추가해야만 사용 가능하다.

### 주석

 - // : 한줄 주석 처리
 - # : 한줄 주석 처리
 - /* */ : 여러줄 주석 처리 (/*: 시작 */: 끝)
